# Case Studies

Five landmark formalisation (or formal-analysis) efforts that the report's `04-case-studies` chapter draws on.

| Case | One-line claim | Tool | Scale |
|---|---|---|---|
| [[Feit-Thompson Odd Order Theorem]] | Every finite group of odd order is solvable - machine-checked for the first time. | Coq (Ssreflect) | ~150,000 LoC, ~6 person-years |
| [[Four Colour Theorem]] | Every planar graph is 4-colourable - previous computer-assisted proof now fully verified. | Coq | 60,000+ LoC |
| [[Gödel Incompleteness Theorems]] | First complete mechanised proof of the second incompleteness theorem (for hereditarily finite sets). | Isabelle/HOL | - |
| [[PFR Conjecture]] | Marton's Polynomial Freiman-Ruzsa conjecture - formalised within 3 weeks of publication. | Lean 4 + mathlib | community effort |
| [[Needham-Schroeder Protocol]] | A 17-year-old "secure" protocol broken by formal analysis (man-in-the-middle attack). | CSP/FDR | - |

Each file below covers: claim · formalisation details (tool, LoC, effort, year) · what it demonstrates for this project · quotes & page refs for the report.
