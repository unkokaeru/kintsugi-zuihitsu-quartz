It should have the title of the seminar.
It should have the name of the seminar's speaker.
It should have flawless spelling.
It should have flawless grammar.
It should have substantial content.
It should be accurate to the seminar.
It should cover the entire seminar's content.
It should state the final seminar take-home-message.
It should cover the societal context of the seminar.
It should cover the research context of the seminar.
It should be very engaging for the reader.
It should be accessible to a lay audience.
It should cover external sources, evidenced by 1+ reference(s) and 1+ quoted opinion(s).
All external sources should be relevant to the seminar.
All external sources should either balance or support the seminar.
