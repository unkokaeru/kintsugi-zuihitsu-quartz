# One Two Skip a Few

|                                                                                                                                                                                                                                                                                                                                                             |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ## One, Two, Skip a Few                                                                                                                                                                                                                                                                                                                                     |
| Create a function which calculates how many numbers are missing from an **ordered** number line.                                                                                                                                                                                                                                                            |
| This number line starts at the first value of the array, and increases by 1 to the end of the number line, ending at the last value of the array.                                                                                                                                                                                                           |
| ### Examples                                                                                                                                                                                                                                                                                                                                                |
| ```<br>howManyMissing([1, 2, 3, 8, 9])<br>output = 4<br># The numbers missing from this line are 4, 5, 6, and 7.<br># 4 numbers are missing.<br>howManyMissing([1, 3])<br>output = 1<br><br>howManyMissing([7, 10, 11, 12])<br>output = 2<br><br>howManyMissing([1, 3, 5, 7, 9, 11])<br>output = 5<br><br>howManyMissing([5, 6, 7, 8])<br>output = 0<br>``` |
| ### Notes                                                                                                                                                                                                                                                                                                                                                   |
| If the number line is complete, or the array is empty, return `0`.                                                                                                                                                                                                                                                                                          |
