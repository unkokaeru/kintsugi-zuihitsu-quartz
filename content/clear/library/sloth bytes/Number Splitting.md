# Splitting Up Numbers

Create a function that takes a number `num` and returns each place value in the number.

## Examples

```python
num_split(39)
output =[30, 9]

num_split(-434)
output = [-400, -30, -4]

num_split(100)
output =[100, 0, 0]
```
