| ### Dependable Jobs Schedule                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| You're given a number of jobs and a list of dependencies.  <br>Each job is labeled from `0` to `jobs - 1`.  <br>Each dependency `[a, b]` means **job** `a` **can only start after job** `b` **is finished**.                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Return `true` if all jobs can be finished, or `false` if there's a circular dependency.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ```<br>finishAll(2, [[1, 0]])<br>output = True<br><br>finishAll(2, [[1, 0], [0, 1]])<br>output = False<br>## job 1 depends on job 0<br>## job 0 also depends on job 1<br>## → circular dependency, cannot complete either job<br><br>finishAll(3, [[1, 0], [2, 1]])<br>output = True<br>## job 0 → job 1 → job 2<br>## no cycles, all jobs can be finished in order<br><br>finishAll(1, [])<br>output = true<br>## only one job (0) with no dependencies<br>## → can be completed immediately<br><br>finishAll(11, [[6, 10], [4, 3], [9, 2], [2, 3], [6, 1], [2, 8], [10, 1], [10, 2], [5, 3], [0, 10], [7, 4], [6, 1]])<br>output = true<br>``` |
