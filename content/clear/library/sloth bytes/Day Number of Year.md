| ## Day Number of Year                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| You're given a date string in the format **month/day/year**, based on the **[Gregorian calendar](https://link.mail.beehiiv.com/ss/c/u001.3nbWF_VfoyqHMq1hZfbT-NG3FHKdiVkHrh6yttKjCafDgko3nirkl9kKTtEjCd6-hg4F9txO3bhm2cgA7HrVkMpZOEpcysGTcmy5XEb5Vw_9pEnAdgWtqeggHNgMt71LL8cV8fVJQ6Ke1BuAwQqTA5MwdT8cFtxxQMGopNCoJf4KlonTaRgq_V2vH7h7MrSgJMF-H3pHaHYuCfW6u8saO-25UjlhbbslXr01P64HDrKXbfqGrfdeCgvZWtuFPKCqkDMIIX3X3gldzD9lzaqCQA/4m3/a4G3_Q19Qemw7gdRRil3QQ/h11/h001.ZLDkEyrRV8X00PBTYVMa3dUQIircLPKxl37YOV6PpTI)**. Your task is to return **which day of the year** that date corresponds to (1–365, or 1–366 for leap years). |
| ### Examples                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ```<br>dayOfYear("12/13/2020")<br>output = 348<br><br>dayOfYear("11/16/2020")<br>output = 321<br><br>dayOfYear("1/9/2019")<br>output = 9<br><br>dayOfYear("3/1/2004")<br>output = 61<br><br>dayOfYear("12/31/2000")<br>output = 366 # leap year<br>dayOfYear("12/31/2019")<br>output = 365 #non leap year     <br>```                                                                                                                                                                                                                                                                                                           |
| ### Notes                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| - Make sure to account for **leap years** when February has 29 days.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
