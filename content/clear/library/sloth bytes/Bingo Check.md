| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ## **Bingo Check**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Create a function that takes a 5x5 2D list and returns `True` if it has at least one Bingo, and `False` If it doesn't.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ### **Examples**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ```<br>bingo_check([<br>  [45, "x", 31, 74, 87],<br>  [64, "x", 47, 32, 90],<br>  [37, "x", 68, 83, 54],<br>  [67, "x", 98, 39, 44],<br>  [21, "x", 24, 30, 52]<br>])<br>output = True<br><br>bingo_check([<br>  ["x", 43, 31, 74, 87],<br>  [64, "x", 47, 32, 90],<br>  [37, 65, "x", 83, 54],<br>  [67, 98, 39, "x", 44],<br>  [21, 59, 24, 30, "x"]<br>]) <br>output = True<br><br>bingo_check([<br>  ["x", "x", "x", "x", "x"],<br>  [64, 12, 47, 32, 90],<br>  [37, 16, 68, 83, 54],<br>  [67, 19, 98, 39, 44],<br>  [21, 75, 24, 30, 52]<br>]) <br>output = True<br><br>bingo_check([<br>  [45, "x", 31, 74, 87],<br>  [64, 78, 47, "x", 90],<br>  [37, "x", 68, 83, 54],<br>  [67, "x", 98, "x", 44],<br>  [21, "x", 24, 30, 52]<br>]) <br>output = False<br>``` |
| ### **Notes**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Only check for diagonals, horizontals and verticals.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
